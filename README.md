# 🎨 Happy Holi Balloon Animation

An interactive **Happy Holi web animation** built using **HTML, CSS, and JavaScript**.
Colorful balloons float on the screen and when you **click a balloon it bursts and releases Holi color particles (gulal effect)**.

This project is a small **fun festive animation** that demonstrates **DOM manipulation, CSS animations, and JavaScript event handling**.

---

## ✨ Features

* 🎈 Floating animated balloons
* 💥 Click a balloon to burst it
* 🎨 Color particle explosion (Holi gulal effect)
* 🌈 Animated gradient background
* 📱 Responsive centered text
* ⚡ Pure HTML, CSS, and JavaScript (no libraries)

---

## 🛠 Technologies Used

* HTML5
* CSS3 (Animations & Keyframes)
* JavaScript (DOM & Events)

## 🎯 How It Works

1. Balloons are dynamically created using JavaScript.
2. They float upward using **CSS animations**.
3. When a balloon is clicked:

   * It is removed from the screen.
   * Color particles are generated.
   * Particles move outward creating a **Holi color burst effect**.
